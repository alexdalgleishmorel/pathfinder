# Pathfinder

## Setup
- Clone the repository on your local PC.
- Navigate into `pathfinder`.
- Run `npm install` to install the dependencies.
- Run `npm install -g @angular/cli` to install the interface that will be used to run the program.

## Running

- Run `ng serve`. ***This command can take some time to execute, wait until the terminal displays `âœ” Compiled successfully`.
- Navigate to `http://localhost:4200/`.
